EPUB Settings

Global

  • Justify text: whether you want text to be fully justified while reading.
  • Hyphenate text: whether you want text to be hyphenated while reading.
  • Fast theme switching: enables tapping with two or three fingers anywhere on the page to cycle through your reading color themes (text and paper colors). Two fingers cycle forward, three fingers cycle backward. This is useful if you want to quickly switch themes without opening any menus.
  • Multi-color bookmarks: bookmarks in Marvin are colored red. If you enable this, you’ll be able to create bookmarks with different colors (red, green, or blue). You might use differently colored bookmarks to represent different kinds of annotations.
  • Scrubber behaviour: set to ‘section’ mode to view your position and scroll within the current chapter or section in the book. Set to ‘book contents’ mode to view your position and scroll across chapters or sections in the table of contents.

Pagination mode

(settings applicable while you’re reading in page viewing mode)

  • Page turn style: the kind of animation Marvin should use when turning pages.
  • Page turn speed: the speed of the animation when turning pages. This is not applicable if the page turning style is set to instant.
  • Page accessories: determines what is shown in the headers and footers of a page.
  • Headers and footers: tells Marvin how you’d like page accessories to show (e.g. shown dim or prominently).
  • Vertical margin size: how wide or narrow you want the vertical page margin to be (the horizontal margin size is set in the book’s formatting menu because that’s a per-book setting). On smaller devices such as iPhones, if you set the vertical margin size to small there might not be space to show headers and footers.
  • Show clock in header: whether to show the time in the page header.
  • Vertical pan for brightness: allows you to pan vertically on the screen to adjust brightness up and down.
  • Gestures: determines what page taps and swipes do (e.g. tapping on the right to move to the next page).

Vertical scrolling mode

(settings applicable while you’re in vertical scrolling mode)

  • Edge tapping: whether tapping on the left, right, and bottom screen edges scrolls a page forward. Tapping on the top screen edge scrolls you a page back.
  • Guide bar on advance: whether a guide bar is shown to help you track your position when advancing using edge tapping. Only applies when edge tapping is enabled.
  • Bottom progress bar: shows a thin progress bar at the bottom of the screen.

Text selection menu

(this is the menu that appears when you tap and hold to select a word or phrase in a book)

  • Deep view: shows Marvin’s Deep View function to find charaters, places, and events in your book, make intelligent web and Wikipedia searches about them, and also build summaries of your book. This is very cool (right now, this is available for books written in English).
  • Define/dictionary: defines the selected text in the dictionary.
  • Share: shares the selected text using iOS’ share sheet.
  • Textshot: creates a poster from the text your have selected (maybe a quote) and lets you share it. Similar to image textshots you see on Twitter.
  • Highlight: highlights the selected text.
  • Highlight and annotate: highlights the selection and lets you annotate it.
  • Search: searches for the selected text in your book.
  • Wikipedia: looks up the selected text in Wikipedia.
  • Custom commands: lets you create your own commands to link selected text to external apps or web services (e.g. send text to Evernote, or you favourite Twitter app).

Custom fonts

  • Manage: lets you load custom fonts into Marvin to use in your books. Font files must reside in Marvin’s documents directory for Marvin to see them.

Text-to-speech

  • Follow paragraphs and chapters: have paragraphs and chapters scroll with the text that is being spoken.
  • Speak in voice: choose the voice that Marvin should use when reading books aloud. When this is set to ‘set by book’, Marvin will automatically select the most appropriate voice to use for the book.
